# Data Engineering Fundamentals

## Key Concepts: The Data Warehouse Toolkit

Focus on dimensional modeling with these core concepts:
- **Fact Tables** - Store quantitative business metrics
- **Dimension Tables** - Store descriptive attributes
- **Grain** - Level of detail in a fact table
- **Star Schema** - Efficient data warehouse design pattern
